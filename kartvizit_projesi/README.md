#  Dijital Kartvizit Projesi

Bu proje, temel web teknolojileri kullanılarak oluşturulmuş, modern tasarımlı ve dinamik özelliklere sahip bir dijital kartvizit uygulamasıdır. Kullanıcıyı günün saatine göre farklı mesajlarla karşılayan kişisel bir tanıtım sayfasıdır.

# Özellikler
* Dinamik Karşılama: JavaScript kullanılarak günün saatine (sabah, öğle, akşam) göre değişen özel mesajlar.
* Modern Tasarım: CSS ile oluşturulmuş cam efekti (Glassmorphism) ve yumuşak geçişli arka plan (gradient).
* Responsive Tasarım: Mobil, tablet ve masaüstü cihazlarla tam uyumlu (Media Queries).
* Sosyal Bağlantılar: LinkedIn, GitHub ve E-posta için ikonlu, etkileşimli butonlar.

# Kullanılan Teknolojiler
* HTML5: Sayfa yapısı ve içerik.
* CSS3: Görsel tasarım, animasyonlar ve responsive düzen.
* JavaScript: Dinamik zaman tabanlı mesaj sistemi.
* Font Awesome: Sosyal medya ve iletişim ikonları.
